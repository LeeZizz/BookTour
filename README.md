Visita
Tours bring tourists experiences about Vietnamese culture and people.
Features
• User authentication and account management registration
• Interactive menu experience and quick response
• Tour booking options according to many locations in the country
• Tour schedules suggested by Visita for tourists to book tours
• Reasonable and secure payment process
• Tour tracking function, chat exchange with Visita
• Comprehensive tour booking history with suggestions for next tours
• User-friendly design, easy to use and simple process when booking tours
Technologies
• Frontend: HTML, CSS, Java
• Backend: Node.js with Express.js
• Template Engine: EJS (Embedded JavaScript)
• Database: MySQL
