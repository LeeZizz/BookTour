# Visita

A modern Vietnamese tour-booking platform that brings tourists authentic experiences about Vietnamese culture and people.

## Features

- User authentication and account management registration
- Interactive menu experience and quick response
- Tour booking options for many locations across the country
- Tour schedules suggested by Visita for tourists
- Reasonable and secure payment process
- Tour tracking function and chat support with Visita
- Comprehensive tour booking history with recommendations for next tours
- User-friendly, simple, and easy-to-use design when booking tours

## Technologies

- Frontend: HTML, CSS, Java
- Backend: Node.js with Express.js
- Template Engine: EJS (Embedded JavaScript)
- Database: MySQL
